4. Open a file in `vi`, insert text, save, and quit.

vi file.txt  // opens the file file.txt in the Vi editor

In Normal Mode:
- i ‚Üí Insert before cursor
- a ‚Üí Append after cursor
- o ‚Üí Open new line below
- Esc ‚Üí Return to Normal mode
In Command-line Mode (:):
- :w ‚Üí Save (write)
- :q ‚Üí Quit
- :wq ‚Üí Save and quit
- :q! ‚Üí Quit without saving

7. Write a shell script that accepts two numbers and prints their sum, difference, and product.

nano file1.sh   // will create a shell script


<------------inside file1.sh--------------->

#!/bin/bash

# Prompt for input
echo "Enter first number:"
read num1

echo "Enter second number:"
read num2

# Calculate and display results
sum=$((num1 + num2))
diff=$((num1 - num2))
prod=$((num1 * num2))

echo "Sum: $sum"
echo "Difference: $diff"
echo "Product: $prod"

#!/bin/bash

# Prompt for filename
echo "Enter the filename to check:"
read filename

# Check if file exists and is readable
if [ -e "$filename" ]; then
    if [ -r "$filename" ]; then
        echo "The file '$filename' exists and is readable."
    else
        echo "The file '$filename' exists but is NOT readable."
    fi
else
    echo "The file '$filename' does NOT exist."
fi

5. Write a script using `case` to check if a given character is a vowel or consonant.

<------------inside file1.sh--------------->

#!/bin/bash

# Prompt the user for a character
echo "Enter a single alphabet character:"
read char

# Convert to lowercase (optional, for case-insensitive match)
char=$(echo "$char" | tr 'A-Z' 'a-z')

# Check if it's a single alphabet character
if [[ ! "$char" =~ ^[a-zA-Z]$ ]]; then
  echo "‚ùå Please enter a valid single alphabet character."
  exit 1
fi

# Use case to check for vowel or consonant
case "$char" in
  a|e|i|o|u)
    echo "‚úÖ '$char' is a vowel."
    ;;
  *)
    echo "üî§ '$char' is a consonant."
    ;;
esac
